论流体液柱从上向下流向圆柱体时的变化

当一流体液柱从上向下流向一较宽（注意）圆柱体时，在流体与圆柱体碰撞点的周围会出现一层薄“水膜”，且停止流水后，水膜的两端则会向中间收缩。

公式中所涉及的字母及其含义：L:水膜长度，v：水流速度，t:沿圆柱体流一周所用的时间，d：膜厚，π：圆周率，r：圆柱体半径，V：水膜体积（单位无特殊规定）。

推导思想：水膜的长度就是，如果把水膜看成圆柱体（水膜是空心圆柱体）该圆柱体的高，利用一些关于圆柱体的公式即可求出水膜长的表达式。

推导过程：因为水膜是空心圆柱体，所以水膜的体积为整体的体积减去水膜内圆柱体体积，即
V=π（r＋d)²L-πr²L
化简后得：V=L[dπ（d＋2r)]

所以L=V/dπ（d＋2r)

因为水膜体积中含有L，所以必须以另一种方式表示体积，而水流速度乘以沿圆柱体流一周所用的时间可以满足，即V=vt。

将其带入到含L方程中，即可得出正确方程：L=vt/dπ（d＋2r) 意义：说明了流体液柱从上向下流向圆柱体时，圆柱体上的液体并不与液柱等宽，其长度将改变为L，可能会用于工业。

附：本人于08-09游泳时捡到一个塑料杯，于是用它舀水冲泳池边的圆杠，意外发现该原理并想出方程雏形（错误），经过推导后得出正确方程，并以此记录。
